# Upgrad Frontend Mini Project

A static, responsive frontend built with **HTML5** and **CSS3** as part of the Upgrad Frontend course.  
This project demonstrates professional layout design and structured styling without JavaScript.

---

## Project Overview

This repository contains a multi-page frontend web project created using semantic HTML and external CSS.  
It highlights clean UI design, consistent styling, and navigation across pages.

---

## Technologies Used

- HTML5
- CSS3

_No JavaScript or backend logic included._

---

## Getting Started

1. Clone the repository:
   git clone https://github.com/Harshith917/Upgrad_Frontend_Mini_Project.git
2. Navigate to the project folder.
3. Open `index.html` in your browser -> Open with Live Server, if extension is present(simple way).
